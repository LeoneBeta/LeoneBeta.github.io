Scapy

O primeiro passo é saber por onde seus pacotes passam, assim foi utilizado a biblioteca SCAPY da linguagem python para fazer a criação dos pacotes personalizados onde temos o controle de todas as informações dentro deles, incluindo o TTL (Time To Live), responsável pelo número de "saltos" possível que o pacote pode dar, assim, temos como resposta desses envios, os roteadores por todo o caminho entre sua localização atual e o IP de destino fornecido no site.

group

API IPInfoDB

Essa API recebe como parâmetro os IPs adquiridos pelo SCAPY então retorna dados geográficos a espeito desses IPs, como, país, estado, cidade e coordenadas geograficas. Esses dados são tratados e separados pois pode ocorrer que alguns roteadores encontrados pelo SCAPY estejam ocultos, então seus dados de localização serão retornados como vazio.

settings

MY API

Após o tratamento do resultado da API IPInfoDB, esses dados precisam ser passados para nosso site, uma aplicação simples utilizando HTML, CSS e JS, então utilizando a biblioteca Flask que nos possibilita criarmos nosssa própria API, a utilizamos para que a aplicação web envie sua requisição contendo o IP de destino para nossa API, que o retorna com os IPs encontrados pelo caminho e suas respectivas localizações.

Aplicação Web

Nossa Aplicação Web Trabalha utilizando uma API de geração de mapa para dar corpo ao projeto, fazemos a requisição a nossa API enviando um IP de destino desejado para busca e retorna os dados dos roteadores entre sua localização atual e o destino solicitado, assim a aplicação web faz o tratamento dessa entrada utilizado os dados para traçar no mapa a rota onde o pacote percorreu.
